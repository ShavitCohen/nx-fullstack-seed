# seed

This is a seed project created with NX [Nx](https://nx.dev).
It contains:

- Next.js App
  - Netlify CMS
  - MDX integration
  - TailwindCss
- Storybook
- Node.JS
  - Celebrate + Joi
- Husky

```
Dev:
nx storybook ui-components
nx serve app
nx serve backend

Custom Generators (Example):
nx workspace-generator new-article

```
